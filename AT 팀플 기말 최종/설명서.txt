1. 첫 화면에서 오른쪽 페이지를 클릭하면 시작
1-2. 키보드에서 1-8을 통해 원하는 장면으로 이동 가능
2. 용궁 배경이 나오면 용궁 입구 클릭
3. 검은색 배경이 다 내려오고 초록색 사각형이 나타나고 사각형으로 마우스 위치를 이동
4. 마우스를 화면 위로 이동(거북이가 바다를 나가는 장면을 표현)
5. 용궁으로 들어가려면 마우스를 용궁 입구에 대고 좌클릭
6. 용왕이 "토끼를 가두거라" 라는 메시지를 남기면 다음 화면에서 스페이스를 클릭(철창이 떨어짐, 안 눌러도 다음 화면으로 넘어감)
7. 마우스를 화면 위로 이동시키고 몇초 기다리면 다음 화면으로 넘어감
8. Press Right라는 텍스트가 뜨면 키보드 오른쪽 화살표 버튼을 이용해 화면을 넘김
8-2. TTS에 맞게 오른쪽 화살표 버튼을 누르면 다음 장면으로 계속 이어감(이때 너무 빨리 오른쪽 버튼을 클릭하면 안됨)
9. "신하를 용서해주시옵소서" 라는 메시지가 끝나고 오른쪽 버튼을 누르면 2개의 선택지가 나옴.
9-2. 오른쪽 버튼을 한 번 누르면 자라가 바위에 머리를 박고 곧 다음 장면으로 넘어감. 다음 장면으로 바뀌고 오른쪽 버튼을 계속 누르다 "How tragic"이라는
텍스트가 나오면 오른쪽 아래의 back 버튼을 통해 선택지로 돌아갈 수 있음
9-3. S를 누르면 새로운 엔딩으로 이어짐
10. 신선이 내려오고 자라가 놀라면 신선보다 약간 왼쪽 아래를 좌클릭하면 다음 장면으로 이동
11. 문장이 끝날때마다 좌클릭
12. 자라랑 신선이 다 움직이면 다시 한 번 좌클릭
13. How to Play책을 누르면 설명서가 나오고 옆에 화살표 버튼을 통해 책에서 나갈 수 있음. 자라를 누르면 게임 시작
14. 
아무것도 누르지 않고 Mix를 누르면 아무것도 안 나옴
올바른 레시피를 섞지 않으면 잘못된 물약이 나온다 뜸
책을 열어서 레시피를 확인 가능함
잘못된 레시피를 만들거나 아무것도 만들지 않아도 Reset 버튼을 통해 리셋 가능함
초록색 물약을 냄비로 드래그 하고 맨드레이크(2번째 약재)를 드래그 시키고 Mix를 누르면 용왕을 변신시키는 약이 생성됨 T를 눌러 다음 장면 이동 가능
노란색 물약을 냄비로 드래그 하고 노란색 꽃을 냄비로 드래그 시키고 Mix를 누르면 용왕을 살리는 약이 생성됨 S를 눌러 다음 장면 이동 가능
15.(transform 물약을 만들었을때)거북이가 용왕 앞에서 멈추면 자라를 클릭. 화면이 전환되면 용왕을 클릭하고 오른쪽 아래의 back 버튼을 통해
다시 약을 제조할 수 있는 화면으로 넘어감
16.(save 물약을 만들었을때)거북이가 용왕 앞에서 멈추면 자라를 클릭. 화면이 전환되면 용왕을 클릭하고 다른 물약을 만들고 싶다면 오른쪽 아래의
back button을 클릭, 엔딩을 이어가고 싶으면 next 버튼 클릭
17. next버튼을 누른뒤 장면이 바뀌면서 크레딧이 나타남. 상하좌우 버튼을 통해 자라를 움직일 수 있음. 자라가 엄지척 하는 장면이 나오면 코드가 끝남
